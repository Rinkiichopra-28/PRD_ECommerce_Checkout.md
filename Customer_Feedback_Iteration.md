#  Post-Launch Review & Iteration Plan

## 1. Early Signals (30-Day Post Launch)
* **Cart Abandonment:** Reduced from **68% to 55%** (Achieved 13% out of target 15% reduction).
* **Express Pay Adoption:** 42% of mobile guest users chose Apple Pay / Google Pay.

## 2. Unanticipated User Friction
* **Issue:** Users entering promo codes found the input box hard to find on mobile guest view.
* **Iteration Plan (V1.1):** Expand promo code drawer by default when cart total exceeds threshold.
